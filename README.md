## Cara Menjalankan

1. Pastikan Flutter sudah terinstall dan macOS desktop support sudah diaktifkan:
```bash
flutter config --enable-macos-desktop
```

2. Jalankan aplikasi:
```bash
flutter run -d macos
```

3. Atau build aplikasi untuk distribusi:
```bash
flutter build macos
```
